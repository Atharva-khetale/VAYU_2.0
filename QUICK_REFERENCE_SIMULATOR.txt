================================================================================
                        TROUBLESHOOTING
================================================================================

Issue: "No module named 'simulator'"
Fix:   cp simulator.py VAYU_X_SYSTEM/
       cd VAYU_X_SYSTEM
       python simulator.py

Issue: "ImportError: No module named 'numpy'"
Fix:   pip install numpy

Issue: "simulation runs but no output"
Fix:   Press Ctrl+C to stop
       Try: python simulator.py 2>&1 | head -50

Issue: Slow performance with many robots
Fix:   Reduce robot count: VAYUSimulator(num_robots=1)
       Or increase timestep: simulator.dt = 0.1

Issue: "port already in use" error
Fix:   You don't need a port! No web server needed.
       This is pure Python simulator.
