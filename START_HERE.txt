================================================================================
                        WHAT EACH AGENT DOES
================================================================================

Vision Agent (vision_001):
  - Detects persons in camera feed
  - Uses TensorFlow Lite for inference
  - Outputs person locations & confidence
  - Simulation: Generates realistic detections

Voice Agent (voice_001):
  - Detects wake words ("vayu")
  - Recognizes voice commands
  - Text-to-speech responses
  - Simulation: Random keyword detection

Motion Agent (motion_001):
  - Controls robot movement
  - Tracks position (x, y, heading)
  - Executes navigation commands
  - Simulation: Realistic movement patterns

Safety Agent (safety_001):
  - Monitors battery voltage
  - Tracks temperature
  - Detects obstacles
  - Triggers emergency stop
  - Simulation: Realistic sensor readings

Mission Planner (mission_001):
  - Queues and executes missions
  - Plans navigation paths
  - Coordinates other agents
  - Makes high-level decisions
  - Simulation: Default museum tour

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: Port 5000 already in use?
Solution: python main.py --port 8080

Problem: Dependencies won't install?
Solution: pip install --upgrade pip
          pip install -r requirements.txt

Problem: Dashboard won't load?
Solution: curl http://localhost:5000/health
          (Check if Flask is running)

Problem: Agents not starting?
Solution: python -c "from config import validate_config; print(validate_config())"

Problem: Need debug output?
Solution: LOG_LEVEL=DEBUG python main.py --interactive
